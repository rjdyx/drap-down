项目运行顺序：
1、先在项目根目录下打开命令行，执行npm install 下载js依赖

2、执行npm run dev进入开发模式，访问localhost:8080

3、执行npm run build构建最终的js文件和css文件，生成的文件在项目根目录下的dist文件夹下

说明：1、最终生成的js文件可直接使用，drop-down.[hash].js文件为本拖动项目的js插件代码，由于代码会经常变动，故添加hash，方便修改代码后替换浏览器上的旧缓存。
2、根目录下dist文件夹里的文件放到服务器上可直接访问，在本地直接打开会报路径引用错误，找不到js、css和images，可手动修改引入路径。
3、根目录下example文件夹里的文件可本地直接打开。
